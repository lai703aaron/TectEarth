# TectEarth
GeoDrift: 3D Earth Evolution Simulator  Simulate 250 million years of plate tectonics, from Pangea to today. Control the timeline and use the integrated AI Time-Travel Guide for real-time insights and image generation of ancient eras.  GeoDrift：3D 地球演變模擬器  模擬地球過去 2.5 億年（從盤古大陸到現代）的板塊構造漂移。控制時間軸，並利用內建的 AI 時空導遊，即時取得地質資訊，並生成古代紀元的影像。
